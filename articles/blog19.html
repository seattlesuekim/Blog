date: 2014-07-25 title: Finding an Open Source Project

Looking for an open source project to contribute to is so simple, even I could do it! I just looked at the “explore” section on Github, where I quickly found projects that I use regularly and have noticed bugs. Then, I checked to see how active the project was, and if there are issues that worth looking into.

    Some projects that came up in my search were:
  • Reddit
  • Rubocop
  • Kathy Butler
  • Ideavim (I use this every day)
  • One Bus Away for Windows Phone (I use this every day)

The plan is to start from the top and if it isn’t feasible to contribute, then to move on. More updates coming soon!
