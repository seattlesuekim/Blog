date: 2014-01-17 title: Ideavim

Vim is for cool people. It's been around since the 80's, so it's vintage. If you like tweed, moustaches and highly customizable text editors, you'll love Vim.

I use IntelliJ IDEA as my primary IDE. Now I'm using a Vim emulator plugin called Ideavim. Someday soon, I will form the new habits of keeping my hand off the track-pad, as well as staying in command mode when not typing any code. With the help of a free book called Practical Vim, as well as this awesome vim cheatsheet, I am on track to be Vimming with the pros.

More importantly, this is the week I started learning Javascript. Jeff Lembeck took an entire work week to teach us Javascript. He got us excited about using a programming language which bring the web into the 21st century. Based on a recommendation from Martha Kelly, I have begun reading Douglas Crockford's "Javascript: the Good Parts." In the upcoming week, I'll be on a team which will implement a RSS feed. I'm looking forward to using some cool authentication technology.
